Cette application affiche une carte de visite et un QR code. D'autres personnes peuvent ajouter votre numéro dans leur application de contacts en scannant le QR code.

Il est possible de configurer trois profils indépendants : privé, entreprise et autre. Les noms de profil sont seulement des suggestions. N'hésitez pas à les modifier dans les paramètres.

Vous pouvez partager les informations suivantes :
• Nom
• Numéro de téléphone
• Courriel
• Adresse
• Entreprise
• Titre
• Site Web

Pourquoi utiliser cette application :
• Material Design
• Pas de publicité ni de ciblage
• Entièrement open-source : https://github.com/characterdog/share-my-number

Aidez à traduire cette application : https://crowdin.com/project/share-my-number
