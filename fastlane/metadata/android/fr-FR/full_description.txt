Cette application affiche une carte de visite et un QR code. D'autres personnes peuvent ajouter votre numéro dans leur application de contacts en scannant le QR code.

Il est possible de configurer trois profils indépendants : privé, entreprise et autre. N'hésitez pas à modifier les noms dans les paramètres.

Les informations suivantes peuvent être partagées :
• Nom
• Numéro de téléphone
• Courriel
• Adresse
• Entreprise
• Titre
• Site Web
